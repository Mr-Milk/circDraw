Stand alone programs for extending the power of circDraw service

CircDraw on local machine

We not only provide service through web browsers, stand alone programs like python package and command line interface can also facilitate users epecially if they don't have a GUI on their server.

CircDraw stand alone python package has two main functions:

  • Visulizing circular RNA with emphasis on backsplicing sitea, internal structures and epi/genetic information.

  • Command line interface $ circDraw-upload for upload files to use circDraw analysis webservice.

Install: Following the instructions on our github repo to install these tools on your computers.


Data source

Human Genome Data: Gencode

circRNA Data: circRNADb , CIRCpedia
